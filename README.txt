DbExtensions
=============================================================================== 

DbExtensions consists of basically 3 components that can be used together 
or separately: 

1. A set of extension methods that simplifies raw ADO.NET programming
2. A dynamic SQL API
3. A Micro-ORM that integrates with System.Data.Linq.Mapping

This project was originally hosted at SourceForge, source code and releases for
versions 1.x, 2.x and 3.x are available at

  https://sourceforge.net/projects/dbextensions/

Source code and releases for version 4.0 onwards are hosted at

  https://github.com/maxtoroq/DbExtensions

Also available via NuGet

  http://www.nuget.org/packages/DbExtensions

To run the samples against SQL Server Compact and SQLite download the
Northwind_embedded.zip file and unzip in samples/Northwind
